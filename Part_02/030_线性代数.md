# 线性代数
*向量、矩阵与变换*

为何同一套数学能描述航天器旋转、电影推荐、量子粒子、市场均衡？答案是线性代数——一套抽象到几乎无所不包，又具体到驱动现代技术底层运算的理论。

线性代数研究的是*线性*结构。在这类系统中，输入加倍则输出加倍，输入相加则输出相加。这听起来有局限，但自然界充满了线性现象。即使在非线性世界，线性代数也提供最佳的局部近似——微积分告诉我们，光滑函数在任意点附近都可以用线性函数很好地逼近。

**核心洞见**：每个线性变换都是视角的转换——系统性地重塑空间，同时保持基本结构不变。掌握这种思维，你就获得了一门通用语言，足以描述几乎所有定量学科中的变化与关系。

## 线性的力量

一个函数 f 是线性的，需满足两条：
1. **可加性**：f(x + y) = f(x) + f(y)
2. **齐次性**：f(cx) = c·f(x)

看似简单，但威力巨大。

**叠加原理**：在线性系统中，对多个输入的响应等于对每个输入单独响应之和。这是电路分析、量子力学、信号处理的基础。

**分解性**：任何输入都可以分解为更简单的部分，分别分析后重新组合。复杂问题由此可控。只要知道线性函数在几个基本构件上的作用，就能确定它对任意输入的作用。非线性函数没有这样的保证。

**可计算性**：线性方程可以系统地求解。我们有能处理数百万变量的高效算法。

考虑摄氏转华氏：F = (9/5)C + 32。这是线性的吗？不是——那个 +32 破坏了齐次性。10°C 对应 50°F，但 20°C 是 68°F 而非 100°F。单独的 f(C) = (9/5)C *是*线性的：输入加倍，输出也加倍。这个区别意义重大：线性带来可预测性。

## 向量：从箭头到抽象

向量最初是箭头——有大小有方向的量。在平面上，向量 **v** = (v₁, v₂) 是从原点出发的箭头。向量加法遵循"首尾相接"：把第二个箭头的尾接在第一个头上，和向量从最初尾到最终头。数乘是拉伸或压缩箭头。

**物理图像**：一艘船以 3 m/s 向北航行（向量 **v**），水流以 4 m/s 推它向东（向量 **u**）。实际运动是 **v** + **u**——指向东北、速度 5 m/s 的箭头。速度、力、电场都同时有大小和方向，向量运算几何上自然。

但向量远不止箭头。一个包含 n 个数字的列表 (x₁, x₂, ..., xₙ) 就是 n 维空间中的向量。两个 n 元组逐分量相加，数乘对每个分量缩放。这立即把向量推广到三维之外：1000 个特征的数据集是 1000 维向量，一幅图像的像素强度是维数等于像素数的向量，公司 50 项指标的季度财务数据是 50 维向量。

这种视角转换——从几何箭头到抽象数字列表——是线性代数应用广泛的原因。我们无法可视化 1000 维，但可以用同样的法则在其中计算。

更深刻的洞见：向量根本不必是数字。考虑所有次数不超过 2 的多项式：3x² - 2x + 5 这样的式子。多项式可以相加，可以与标量相乘，这些运算满足与向量相同的规则。这些多项式构成**向量空间**。

向量空间是任意集合配备加法和数乘，并满足基本公理：加法交换、结合，存在零元素，存在逆元，数乘满足分配律。一旦验证某个集合满足这些性质，就可以应用线性代数的全部工具。

**更多例子**：区间上的连续函数、齐次微分方程的所有解、所有 m×n 矩阵——只要能相加和数乘，就是向量空间。正是这种抽象，使同一套数学描述物理箭头、代数多项式、微分方程和量子态。

## 维数与基：空间有多大

一组向量 {**v**₁, **v**₂, ..., **v**ₖ} 是**线性无关**的，如果没有任何一个可以表示为其他的线性组合。几何上：平面中两个向量线性无关，当且仅当它们不平行，指向真正不同的方向。它们张成整个平面——通过组合它们可达任意向量。加入第三个向量，若它在前两个张成的平面内，它就没贡献新内容。

**基**是既线性无关又张成整个空间的向量集合。恰好"不多不少"：太少无法张成空间，太多必然相关。空间中每个向量都可以*唯一地*表示为基向量的线性组合。

向量空间的**维数**是任意基包含的向量个数。令人惊叹的是，无论选哪组基，这个数都相同。维数刻画空间的"自由度"——可独立移动的方向数目。ℝ² 维数是 2，ℝ³ 维数是 3，次数不超过 n 的多项式空间维数是 n+1。

一旦选定基，每个向量就有唯一的坐标——将其表示为基向量线性组合时的系数列表。ℝ³ 的标准基由单位向量 **e**₁ = (1,0,0)、**e**₂ = (0,1,0)、**e**₃ = (0,0,1) 构成。向量 (3, -2, 7) 就是 3**e**₁ - 2**e**₂ + 7**e**₃。

但你可以选其他基。同一个几何箭头，在不同基下有不同坐标——用不同数字表示同一个东西。这不是复杂化，而是策略：选对基能把困难问题化为简单问题。

**例子**：分析弹簧连接系统的振动。标准基下，方程耦合在一起难以处理。但存在特殊的基——"简正模态"——其中每个基向量代表纯粹的振动模式。在这组基下，系统解耦：每个模态独立地以固有频率振动。物理没变，但正确的坐标揭示了潜在简洁性。

## 矩阵：变换的语言

如果向量是名词，矩阵就是动词。**矩阵**表示*线性变换*——保持线性结构的向量空间之间的函数。它们可以旋转、拉伸、剪切、投影，但始终把原点映到原点，把直线映成直线。

关键洞见：**线性变换完全由其在基上的作用决定**。只要知道 T 把每个基向量映到哪里，利用线性性就能计算它对任意向量的作用：

T(c₁**v**₁ + c₂**v**₂ + ... + cₙ**v**ₙ) = c₁T(**v**₁) + c₂T(**v**₂) + ... + cₙT(**v**ₙ)

无穷多信息（每个点映到哪里）被压缩为有限信息（基向量映到哪里）。

矩阵编码这些有限信息。m×n 矩阵 A 的第 j 列是第 j 个标准基向量的像。要计算 T(**v**)，进行矩阵-向量乘法：A**v** 等于用 **v** 的分量加权地组合 A 的列。这就是线性性的计算实现。

### 变换的几何

平面上的 2×2 矩阵展示了丰富的几何操作：

**旋转角度 θ**：矩阵 [cos θ, -sin θ; sin θ, cos θ] 逆时针旋转每个向量。

**缩放**：矩阵 [a, 0; 0, b] 水平拉伸 a 倍，垂直拉伸 b 倍。当 a = b 时是均匀缩放，形状不变；不同时形状变形。

**投影到 x 轴**：矩阵 [1, 0; 0, 0] 把每个向量压扁到 x 轴上，所有垂直信息丢失。注意这不可逆——一旦投影无法恢复。

### 复合变换与矩阵乘法

复杂变换——先旋转、再拉伸、再旋转——可通过矩阵相乘组合。如果 A 表示变换 T，B 表示变换 S，那么矩阵乘积 BA 表示复合变换：先应用 S，再应用 T。矩阵乘法的规则正是为满足这一要求设计的。

矩阵乘法满足结合律 (AB)C = A(BC)，反映函数复合的结合性。但通常*不*交换：AB ≠ BA。这合理吗？把书绕 x 轴旋转 90°再绕 z 轴旋转 90°，与先绕 z 轴再绕 x 轴，最终方向不同。变换顺序重要，矩阵乘法顺序也必须重要。

### 逆矩阵与方程组

并非每个变换都可撤销。投影把平面压扁成线，无法恢复——无穷多点映到线上同一点。

方阵 A 有**逆矩阵** A⁻¹，如果 A⁻¹A = I（单位矩阵）。此时 A 表示的变换可逆：应用 A⁻¹ 回到起点。

这直接关联求解线性方程组。A**x** = **b** 问：哪个输入 **x** 在变换 A 下产生输出 **b**？如果 A 可逆，唯一解是 **x** = A⁻¹**b**。

方阵的**行列式**编码关键信息。几何上，|det(A)| 是 A 对面积或体积的缩放因子。关键事实：det(A) = 0 当且仅当 A 不可逆。行列式回答：这个变换压缩了维度，还是保持了维度？

## 内积：度量长度与角度

到目前为止，我们讨论了加法和数乘，但还没涉及度量。要谈论长度、角度、距离，需要**内积**。

ℝⁿ 上的标准内积是点积：⟨**u**, **v**⟩ = u₁v₁ + u₂v₂ + ... + uₙvₙ

这个简单公式蕴含丰富几何信息：
- **长度**：||**v**|| = √⟨**v**,**v**⟩
- **角度**：cos θ = ⟨**u**,**v**⟩ / (||**u**|| ||**v**||)
- **正交性**：**u** ⊥ **v** 当且仅当 ⟨**u**,**v**⟩ = 0

如果向量内积为零，它们**正交**。**标准正交基**由两两正交的单位向量构成。这样的基计算上珍贵：坐标由简单内积给出，长度保持不变。

保持内积的矩阵称为**正交矩阵**。它们的列构成标准正交基。旋转和反射是正交的；拉伸和剪切不是。正交矩阵满足 Q⁻¹ = Qᵀ（逆等于转置），使它们计算高效且数值稳定。

## 特征值：寻找不变方向

在所有可能被变换的方向中，有些特殊：它们可能被拉伸或翻转，但变换后仍指向相同方向。这些**特征向量**揭示变换的内在结构。

矩阵 A 的特征向量是满足 A**v** = λ**v** 的非零向量。标量 λ 称为**特征值**。变换只是将 **v** 缩放 λ 倍——完全没有旋转它。

**几何直觉**：考虑拉伸平面的矩阵，水平向量拉伸 2 倍（特征值 2），垂直向量压缩一半（特征值 0.5）。这些是拉伸的自然"坐标轴"，方向保持不变。

**物理图像**：转动陀螺。尽管运动复杂翻滚，旋转轴始终不变。该轴就是旋转算子的特征向量——变换保持不变的方向。

### 为何特征向量重要

特征向量极大简化变换。如果能找到完全由特征向量构成的基，那么在这组基下，变换由**对角矩阵**表示——特征值排在对角线上，其余都是零。应用变换只是独立地缩放每个坐标。

这种**对角化**使计算简单。想连续应用 A 一百次？在特征基下，只需将每个特征值取 100 次幂。矩阵的幂和指数——在微分方程和物理学中至关重要——变成初等运算。

### 斐波那契与黄金比例

斐波那契数列 Fₙ₊₁ = Fₙ + Fₙ₋₁ 可写成矩阵方程。记矩阵为 A，则 [Fₙ₊₁; Fₙ] = Aⁿ [1; 0]。

A 的特征值是 φ = (1+√5)/2（黄金比例，约 1.618）和 ψ = (1-√5)/2（约 -0.618）。对角化得到封闭形式：

Fₙ = (φⁿ - ψⁿ) / √5

黄金比例的出现非神秘——它是特征结构的直接结果。由于 |ψ| < 1，当 n 很大时 ψⁿ 项可忽略，Fₙ ≈ φⁿ/√5。斐波那契数以 φ 为比率几何增长。

### 主成分分析

在统计学和机器学习中，协方差矩阵的特征向量标识数据中方差最大的方向——"主成分"。它们揭示潜在结构：数据沿哪些方向分布最广？

**例子**：测量人的身高和臂展。这两个测量值相关——高个子往往臂展长。绘制数据形成细长点云。第一主成分沿点云长轴（方差最大方向），大致对应"整体体型"。第二主成分垂直于第一个，捕捉剩余方差——可能是"相对于身高的臂长"。

**PCA** 将高维数据投影到方差最大方向，在保留最重要信息的同时降维。广泛应用于图像压缩、基因组学、金融。

## 对称矩阵的优美结构

**对称矩阵**（A = Aᵀ）具有特别美的性质。**谱定理**保证：所有特征值都是实数，对应不同特征值的特征向量相互正交，存在标准正交特征向量基。

这意味着每个对称矩阵都可以写成 A = QDQᵀ，其中 Q 是正交矩阵，D 是对角矩阵。变换分解为：旋转到特征基，沿各轴缩放，旋转回来。

对称矩阵自然出现在没有优先方向的结构中：物理学的转动惯量、应力张量，统计学的协方差矩阵，优化的海森矩阵，图论的邻接矩阵。谱定理使它们易于处理：我们总能求出特征值（都是实数），总能对角化（正交地），总能理解几何（沿主轴拉伸）。

## 应用：抽象的具体表达

### 机器学习中的词嵌入

现代自然语言处理将词表示为高维空间中的向量。在大规模文本语料上训练后，产生了惊人性质：语义关系变成几何关系。向量运算 "king" - "man" + "woman" ≈ "queen" 从学习得到的表示中自然涌现——纯粹的线性代数捕捉了意义。

### 计算机图形学的变换流水线

三维图形中每种变换——旋转、缩放、投影到屏幕——都表示为矩阵。渲染流水线通过矩阵相乘高效组合变换。一个巧妙技巧：增加额外维度使平移（本身不是线性的！）也可表示为矩阵乘法。在四维齐次坐标中，"先旋转、再平移、再缩放"变成单次矩阵乘法。

### 量子力学的数学基础

量子力学建立在复向量空间上的线性代数之上。物理态是向量；可观测量是厄米算子（矩阵）；测量是特征值问题。量子比特的态是二维复向量空间中的单位向量。多个量子比特态存在于张量积空间中——维数随量子比特数指数增长，这正是量子计算机难以经典模拟的原因。

## 为何线性代数如此有效

**叠加性**：解可相加和缩放产生新解。这使分解成为可能：将复杂问题分解为简单部分，分别求解，然后组合。

**可计算性**：线性方程可系统求解。高斯消元及其改进可处理巨大系统。特征向量、SVD 和矩阵分解都有高效算法，可在大规模并行硬件上运行。

**可逼近性**：在任意点附近，光滑函数近似是线性的——这是微积分的泰勒定理。非线性问题通过牛顿法及相关技术变成线性问题的序列。

**普适性**：向量空间的公理最小却威力强大。任何具有加法和数乘的系统都继承整套框架。

## 延伸阅读

**Axler, *Linear Algebra Done Right*（2024）**：强调概念而非计算的优雅教材。推迟引入行列式以专注线性映射，建立深刻直觉。

**Strang, *Introduction to Linear Algebra*（2023）**：经典计算方法教材，配 MIT 开放课程视频。强调四个基本子空间和实际应用。

**3Blue1Brown, *Essence of Linear Algebra*（视频系列）**：出色的几何可视化使变换栩栩如生。最好与教科书配合，既获得直觉又获得严谨性。

**Trefethen & Bau, *Numerical Linear Algebra*（1997）**：面向实际计算。涵盖现代科学计算背后的算法，实现矩阵计算的必读。
