# 数系
*从计数到复数*

孩子学会计数：一、二、三。但很快他们就会遇到问题：三个苹果减去五个苹果是多少？这个简单的问题在自然数的世界里没有答案。

几千年来，数学的历史就是不断遇到这样的问题、然后扩展数的概念的历史。每次扩展都解决了旧问题，但也带来新的问题。从自然数到复数，这条路径不是任意的——它是被实际需求和内在逻辑共同推动的。

更令人惊讶的是：这些为了解决抽象数学问题而发明（或者说发现？）的数系，最终成了描述物理世界的精确语言。复数最初被叫做"虚数"，被认为是纯粹的数学游戏，如今却是量子力学不可或缺的基础。这是怎么发生的？

## 层级：一个嵌套的扩展

数系形成一个嵌套的层级，每一层都包含前一层并增加新的能力：

$$\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}$$

- **自然数（$\mathbb{N}$）**：让我们能够计数
- **整数（$\mathbb{Z}$）**：让我们能够做减法
- **有理数（$\mathbb{Q}$）**：让我们能够做除法
- **实数（$\mathbb{R}$）**：让我们能够度量连续量
- **复数（$\mathbb{C}$）**：让每个多项式方程都有解

但这不仅仅是"越来越多的数"。区分这些数系的，不是它们包含多少个数，而是它们具有什么**结构性质**——能否做减法、能否排序、是否有间隙、是否所有方程都有解。每个系统都为了获得某些性质而牺牲另一些性质。没有完美的数系，理解为什么，就能看到数学结构的深层约束。

## 自然数：一切的起点

自然数 $\mathbb{N} = \{0, 1, 2, 3, ...\}$ 来自计数。孩子不需要懂数学就能判断"这些玩具够不够分"——一个玩具配一个小朋友，看看有没有剩余。这种一一对应的能力，先于数的概念。

这一观察很深刻。数不是某个神秘的柏拉图实体，而是我们给"两个集合有相同数量元素"这种关系起的名字。"三"就是所有三元素集合共有的东西。

**皮亚诺公理**形式化了计数的本质：
1. 0 是自然数
2. 每个自然数 $n$ 都有后继 $S(n)$（即 $n+1$）
3. 0 不是任何数的后继
4. 不同的数有不同的后继
5. **归纳法**：如果某性质对 0 成立，且每当它对 $n$ 成立时也对 $S(n)$ 成立，则它对所有自然数成立

第五条是关键。它说自然数**恰好**是从 0 开始反复取后继所得到的——不多也不少。没有归纳法，我们无法证明加法交换律、无法证明素数有无穷多个。

从后继运算，我们可以递归定义加法和乘法。然后证明所有熟悉的性质——交换律、结合律、分配律。数学不假设 $2 + 3 = 3 + 2$；它从公理证明这一点。

**局限**：自然数对加法和乘法封闭，但对减法不封闭。$3 - 5$ 在自然数中没有答案。这不是缺陷，而是指向下一次扩展的信号。

## 整数：引入负数

整数 $\mathbb{Z} = \{..., -2, -1, 0, 1, 2, ...\}$ 通过添加负数扩展了自然数。历史上，负数在千年间都有争议。怎么能拥有少于无的东西？希腊人回避它们，中世纪欧洲数学家称它们"荒谬"，连帕斯卡都说从 0 减去 4 是"纯粹的胡说八道"。

实际需求来自商业和会计。如果我欠你钱，表示为负余额是自然的。如果我们追踪沿直线的位置，向左的运动应该抵消向右的运动。概念转变需要把数理解为不仅表示大小，还表示**有向**的量。负数不是"少于无"——它是相反方向上的某物。

从构造角度看，整数可以用自然数对 $(a, b)$ 表示意图中的差 $a - b$。$(5, 3)$ 表示 2；$(3, 5)$ 表示 -2。我们定义 $(a, b)$ 和 $(c, d)$ 等价，当且仅当 $a + d = c + b$（这是 $a - b = c - d$ 的改写，避免使用我们还没定义的减法）。整数就是这些等价类。

**获得了什么**：减法的封闭性。我们可以解 $x + 5 = 3$（答案：$x = -2$）。

**保留了什么**：序关系、离散性（每个整数都有直接的前驱和后继）。

**应用**：电荷、轻子数、重子数——这些守恒量可正可负可零。密码学中的模运算（RSA 加密）在整数环 $\mathbb{Z}/n\mathbb{Z}$ 中进行。

## 有理数：分数的世界

有理数 $\mathbb{Q}$ 是可以表示为分数 $\frac{a}{b}$ 的数（$a, b$ 是整数，$b \neq 0$）。毕达哥拉斯学派相信所有的量都是有理的——任何两个长度都可以用一个公共单位来度量。宇宙在两种意义上都是理性的（rational）。

音乐和谐支持这个信念：八度是 2:1，五度是 3:2，四度是 4:3。美丽的和弦对应简单的整数比。行星运动、建筑比例——一切似乎都可以表示为分数。

有理数是我们遇到的第一个**域**——一个加法、减法、乘法和除法（除以零除外）都总是给出有效结果的数系。它们还有一个惊人的性质：**稠密性**。任意两个有理数之间总有另一个有理数（比如它们的平均值）。不像自然数，有理数没有"下一个"——数轴上的有理点无穷密集。

这似乎暗示有理数完全填满了数轴。毕竟，任意两点之间总有另一个有理点，怎么可能还有间隙？

**无理性的震撼**

毕达哥拉斯的世界观在正方形的对角线上破碎了。边长为 1 的正方形，其对角线长度为 $\sqrt{2}$。这是有理数吗？

假设 $\sqrt{2} = \frac{p}{q}$，其中 $p, q$ 没有公因子。那么 $p^2 = 2q^2$，所以 $p^2$ 是偶数，因此 $p$ 是偶数。设 $p = 2k$，则 $4k^2 = 2q^2$，所以 $q^2 = 2k^2$，这使 $q$ 也是偶数。但 $p, q$ 都是偶数意味着它们有公因子 2——矛盾。

因此 $\sqrt{2}$ 是无理数。

这一发现引发了真正的思想危机。传说揭示这个真理的 Hippasus 被同门淹死了（这可能是杜撰的，但足以说明震撼程度）。有理数中有间隙——数轴上有 $\sqrt{2}$ 应该在的"位置"，那条对角线的长度在几何上确实存在，但没有有理数占据它。

**稠密但不完备**：有理数无处不在，但有漏洞——像一个网眼极细的筛子，看起来密不透风，但水还是会漏。这就是完备性与稠密性的区别。

## 实数：填补间隙

实数 $\mathbb{R}$ 填补了有理数的间隙。我们可以构造一个有理数列趋近于 $\sqrt{2}$：1, 1.4, 1.41, 1.414, ...每一项都是有理数，数列无限接近某个东西——但那个东西不是有理数。

用技术语言说，这是一个**柯西序列**——各项最终会彼此无限接近。在完备空间中，每个柯西序列都收敛到空间内的一个极限。有理数不完备；实数完备。

**完备性**是实数的定义性特征：每个有上界的非空实数集都有**最小上界**（上确界）。考虑所有满足 $x^2 < 2$ 的实数的集合。这个集合有上界（比如 2），所以它有最小上界——那就是 $\sqrt{2}$。在有理数中，相应的集合没有最小上界——不存在最小的有理数超过所有平方小于 2 的有理数。

**为什么完备性如此重要？** 因为它是微积分能够运作的原因：

- **介值定理**：连续函数从负变正，必定在某处穿过零——完备性保证那个"某处"存在。
- **极值定理**：闭区间上的连续函数取得最大值和最小值——完备性保证极值存在。
- **序列收敛**：当我们说序列"收敛"，我们指它趋近于一个极限——完备性保证极限存在。

没有完备性，微积分就会崩塌。会有应该穿过横轴但从不等于零的函数；会有应该收敛但没有极限的序列。物理学中的连续现象——位置、速度、温度、场强——都依赖实数的完备性。

**大小的惊人差异**

有理数是**可数的**——我们可以把它们排成一列，最终包含每一个有理数。把所有分数排列在网格中，沿对角线遍历，就能数遍所有有理数。

实数是**不可数的**。康托尔用对角线论证证明了这一点。假设我们有 0 到 1 之间所有实数的完整列表，写成小数展开：

$r_1 = 0.d_{11}d_{12}d_{13}...$
$r_2 = 0.d_{21}d_{22}d_{23}...$
$r_3 = 0.d_{31}d_{32}d_{33}...$

构造一个新数 $s = 0.s_1s_2s_3...$，其中 $s_n \neq d_{nn}$。这个数在第 $n$ 位小数上与每个 $r_n$ 不同，所以它不在列表中。矛盾。

这意味着"几乎所有"实数都是无理数——事实上，"几乎所有"实数都是超越数（不是任何整系数多项式的根）。$\pi$、$e$、$\sqrt{2}$ 这些我们能命名的数，在实数的海洋中只是稀疏的例外。大多数实数无法被命名、无法被计算、无法通过任何有限手段描述——它们作为逻辑必然性而存在，但超越了显式的描述。

## 复数：完备化代数

几个世纪以来，$x^2 = -1$ 没有解。任何实数的平方都是非负的（正乘正是正；负乘负也是正），所以根本没有实数 $x$ 能满足。

但这种不对称令人困扰。方程 $x^2 = 4$ 有两个解；$x^2 = 2$ 有两个解；为什么 $x^2 = -1$ 应该没有解？更实际的问题是：文艺复兴时期的代数学家发现，即使在求三次方程的**实**解时，中间步骤有时也涉及 $\sqrt{-1}$。这些"不可能"的量出现在计算中，然后神秘地消失，留下正确的实数答案。

数学家无法把 $\sqrt{-1}$ 当作无稽之谈而不予理会，因为它出现在有效的计算中。他们必须认真对待它。

**大胆的一步**是定义一个新数 $i$，使其满足 $i^2 = -1$，然后看看接下来会发生什么。

复数具有 $z = a + bi$ 的形式，其中 $a, b$ 是实数。算术自然地运作：

- 加法：$(a + bi) + (c + di) = (a+c) + (b+d)i$
- 乘法：$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$

关键是在需要的地方代入 $i^2 = -1$。

**几何图景**

复数的真正力量来自几何解释。把 $z = a + bi$ 表示为平面上的点 $(a, b)$——**复平面**。实数沿水平轴；纯虚数沿垂直轴。

从这个视角：
- 加法是向量加法
- 模 $|z| = \sqrt{a^2 + b^2}$ 是到原点的距离
- 辐角是从正实轴开始的角度
- **乘法组合模并相加辐角**

关键洞见：**乘以 $i$ 使一个数逆时针旋转 90°**。如果 $z = a + bi$，那么 $iz = -b + ai$。几何上，点 $(a, b)$ 映射到 $(-b, a)$——正是旋转 90°。这就是为什么 $i^2 = -1$：旋转 90° 两次得到 180° 旋转，即取负。

这个几何图景把 $i$ 从神秘的代数技巧变成了自然的几何运算。复数不是实数的任意扩展；它们是**配备了旋转的实数**。

**欧拉公式：深层联系**

数学中最令人惊叹的关系连接了五个基本常数：

$$e^{i\pi} + 1 = 0$$

这源于欧拉公式：$e^{i\theta} = \cos\theta + i\sin\theta$。

为什么这是真的？指数函数、正弦和余弦都有幂级数展开。将 $i\theta$ 代入指数级数，使用 $i^2 = -1$，级数自然分成实部和虚部，恰好对应余弦和正弦。

当 $\theta = \pi$ 时，$e^{i\pi} = \cos\pi + i\sin\pi = -1$。

这不是巧合，而是深刻的结构联系：**增长和旋转是同一回事**，区别只在于指数是实数还是虚数。数 $e$ 源于增长过程；$\pi$ 源于圆；$i$ 源于代数完备化；它们被复数指数联结在一起。

**代数基本定理**

复数是**代数闭**的：每个非常数多项式都有复数根。更精确地说，每个 $n$ 次多项式都完全分解为 $n$ 个线性因子。

这个定理说复数是代数目的的"最终"扩展。实数有间隙（方程 $x^2 + 1 = 0$ 无解）。复数没有。你不能通过要求多项式方程有解来走得更远——每个多项式方程在 $\mathbb{C}$ 中已经有解了。

**代价**：复数不能以与其域运算兼容的方式排序。在任何有序域中，平方必须非负。但 $i^2 = -1 < 0$。这与复数的二维性质有关——序对直线有效，但在平面中没有自然的"大于"。

**获得了什么**：代数封闭性——所有多项式方程都有解。

**失去了什么**：序关系——我们不能说两个复数哪个"更大"。

**量子力学需要复数**

也许最惊人的是：复数对量子力学不仅方便，而且**必需**。量子态是复希尔伯特空间中的向量；概率是复振幅模的平方；薛定谔方程显式包含 $i$：

$$i\hbar\frac{\partial}{\partial t}\Psi = \hat{H}\Psi$$

为什么是复数？这仍然有些神秘。最近的理论和实验工作（2021-2022）表明，量子力学不能只用实数重新表述——实验可以区分"实量子力学"和"复量子力学"，而自然选择了复数。

最初被叫做"虚数"的代数游戏，如今是描述物理实在最基础层次的不可或缺的语言。

## 比较视角

| 性质 | $\mathbb{N}$ | $\mathbb{Z}$ | $\mathbb{Q}$ | $\mathbb{R}$ | $\mathbb{C}$ |
|------|--------------|--------------|--------------|--------------|--------------|
| 对减法封闭 | 否 | 是 | 是 | 是 | 是 |
| 对除法封闭* | 否 | 否 | 是 | 是 | 是 |
| 域 | 否 | 否 | 是 | 是 | 是 |
| 有序 | 是 | 是 | 是 | 是 | **否** |
| 稠密 | 否 | 否 | 是 | 是 | 不适用 |
| 完备 | 否 | 否 | 否 | **是** | 不适用 |
| 代数封闭 | 否 | 否 | 否 | 否 | **是** |
| 基数 | $\aleph_0$ | $\aleph_0$ | $\aleph_0$ | $\mathfrak{c}$ | $\mathfrak{c}$ |

（*除以零除外）

这些权衡是根本性的、不可避免的。有定理表明某些组合是不可能的：
- 没有有序域可以是代数闭的
- 实数是唯一的完备有序域
- 复数是实数的代数闭包

每个系统都牺牲某些东西来获得另一些东西。实数牺牲代数封闭性以换取序和完备性。复数牺牲序以换取代数封闭性。**不存在具有所有理想性质的数系**——理解为什么，就能看到数学结构的深层约束。

## 超越复数？

我们能继续扩展吗？可以，但代价越来越大。

**四元数**（$\mathbb{H}$）是 4 维扩展，乘法不可交换：$ij \neq ji$。Hamilton 在 1843 年散步时发现了它们，并将基本方程刻在石头上：$i^2 = j^2 = k^2 = ijk = -1$。

四元数在表示三维旋转方面极其有用。每次 3D 游戏角色转身时，幕后都在进行四元数乘法。它们避免了"万向节锁定"（欧拉角的问题），广泛用于航空航天、机器人和计算机图形学。

**八元数**（$\mathbb{O}$）是 8 维的，甚至牺牲了结合律：$(ab)c \neq a(bc)$。它们不太常用，但出现在弦理论等领域。

一个定理说这些基本上是唯一的选择。任何实数上的有限维除法代数的维数必须是 1、2、4 或 8。每一步维数翻倍，同时牺牲一个性质：
- $\mathbb{C}$：失去序
- $\mathbb{H}$：失去交换律
- $\mathbb{O}$：失去结合律

没有 16 维的除法代数。规律到此为止。**你不能拥有一切。**

## 扩展的逻辑

回顾数系的演进，我们看到一个清晰的模式：

1. **每个系统解决了前一个系统无法处理的问题**。整数解决减法；有理数解决除法；实数解决极限；复数解决多项式方程。

2. **每次扩展是被迫的，而非任意的**。构造方法不是创造性的发明——它们是在保持现有运算的同时增加新能力的唯一方式。

3. **每次扩展都牺牲了某些东西**。整数失去良基性；实数失去可数性；复数失去序。

4. **终点取决于你的目标**。对于序和完备性，$\mathbb{R}$ 是终点。对于代数封闭性，$\mathbb{C}$ 是终点。不同的目的需要不同的系统。

5. **结构比元素更重要**。使每个系统有用的不是个别的数，而是结构性质——封闭性、序、完备性、代数封闭性。

数系的研究教会我们数学如何成长：面对局限，在保持有效之处的同时谨慎扩展，并发现扩展揭示了某种"一直在那里"的结构。

## 更深的思考

**数是被发现的还是被发明的？**

有两种根本不同的看法贯穿我们的讨论：

**发现观**认为，每个数系一直"在那里"等待被发现。负数、无理数和复数与自然数一样真实。扩展揭示的是预先存在的数学领域，就像探险家测绘一片在他们到来之前就已存在的大陆。

这个观点解释了为什么相同的结构在不同构造中出现，为什么数学如此有效地适用于物理学。如果数学结构是客观存在的，那么物理定律就是在描述那些结构。

**构造观**认为，我们**创造**新的数系来解决问题。当减法失效时，我们将整数构造为有序对的等价类。新的数是人类的发明，尽管受到一致性要求的约束。

这个观点展示了如何从更简单的基础建立严格的数学，并揭开了 $\sqrt{-1}$ 等看似"不可能"的数的神秘面纱。如果我们能构造它们，它们就是合法的数学对象。

两种观点都有洞见。关键是：无论你站在哪一边，数学的结构性约束是客观的。你不能任意发明一个既有序又代数闭的域——定理告诉我们这不可能。数学有自己的逻辑，不管我们是在发现它还是发明它。

**数学不合理的有效性**

物理学家 Eugene Wigner 提出了一个仍然开放的谜题：为什么常常出于纯粹抽象原因发展的数学结构，最终能如此精确地描述物理现实？

复数是为了解代数方程，却成了量子力学的基础。群论是抽象代数，却能预测粒子物理学。实数的完备性看起来像技术条件，却支撑着自然界到处可见的连续现象。

如果数学是发明的，为什么它如此有效？如果数学是发现的，它的本体论地位是什么？这也许是数学哲学中最深刻的问题——至今没有完全令人满意的答案。
