# 量子力学：基础篇
*微观世界的奇异法则*

1909年，Ernest Rutherford 发现原子拥有原子核。这个发现立刻带来一个致命问题。

按照经典电磁学，围绕原子核运动的电子会持续辐射能量，在不到一秒的亿万分之一的时间里螺旋坠入原子核。原子应该瞬间坍缩。

然而你我身体里的每个原子都已稳定存在了数十亿年。

这个矛盾逼迫物理学家走上一条激进的道路：量子力学。他们发现的不是对经典物理的修补，而是彻底的范式更替。在最基本的层面上，宇宙不是由沿确定轨迹运动的粒子组成，而是由某种更奇异的东西——**波函数**——描述的。它编码的是概率，能够同时处于多个状态的"叠加"中,并在测量时以不可预测的方式给出确定结果。

九十多年来,量子力学经受住了所有挑战。它预测的电子磁矩与实验结果吻合到小数点后第十一位——这就像预测从纽约到洛杉矶的距离，误差不超过一根头发丝的宽度。

但它告诉我们关于现实本质的什么信息，至今仍深深不确定。

## 经典图景的崩溃

量子力学是被逼出来的。一系列实验结果显示经典物理根本错了。

**黑体辐射**。加热物体会发光——铁棒从红光变橙光再变白光。但经典物理预测热物体应该发出无穷多能量，这显然荒谬。1900年，Max Planck 发现如果能量只能以离散的包（"量子"）发射，每个包的能量正比于频率 E = hν，就能重现实验结果。他把这当作计算技巧。后来发现这是自然的根本特征。

**光电效应**。紫外光照射金属会打出电子。经典理论说更亮的光应该打出更快的电子。实验却显示：电子速度只取决于光的频率（颜色），而非强度（亮度）。低于某个频率阈值，无论多亮都没用。1905年，Einstein 认真对待 Planck 的量子：光以离散的粒子（光子）存在，每个光子能量 E = hν。电子吸收一个光子，如果能量足够就逃逸出来。这解释了一切。

Maxwell 已经确凿证明光是波（干涉、衍射）。现在 Einstein 说光也是粒子。两种描述都是必需的。

**原子光谱**。氢原子只发射特定频率的光——656 nm 的红线，486 nm 的蓝线——而不是连续谱。Niels Bohr 在1913年提出：电子只能占据离散轨道，在轨道间跃迁时发射或吸收光子。这个模型与观测惊人吻合，但 Bohr 无法解释为什么轨道是量子化的。

**双缝干涉**。让电子一个接一个通过两条狭缝。每个电子在屏幕某一点撞击——粒子行为。但积累成千上万个电子后，它们形成干涉图样——波动行为。如果电子只是简单通过某一条缝，你会看到两条带。干涉图样证明每个电子"知道"两条缝都存在。

更奇怪的是：如果你添加探测器查看电子走哪条缝，干涉图样消失。观测行为改变了结果。

这些现象要求一个新框架。

## 核心框架：一个简单例子

理解量子力学最好的方式是看一个足够简单、可以完全分析的系统：电子自旋。

### 斯特恩-格拉赫实验

1922年，Otto Stern 和 Walther Gerlach 让银原子束穿过非均匀磁场。经典物理预测：原子磁矩可以指向任何方向，所以屏幕上应该看到连续分布。

实际结果：原子束恰好分裂成两个点——向上和向下。磁矩是量子化的。

我们说电子自旋为1/2，沿任何测量轴只能取两个值：上或下。用符号表示为 |↑⟩ 和 |↓⟩。

现在奇异的事情出现了。

先制备沿z轴自旋向上的电子（选择向上的束）。然后测量沿x轴的自旋。你发现50%向右，50%向左——完全随机。

只取向右的电子，再次测量z轴自旋。你可能期望它们仍然是z-上，因为一开始就是。但不——又是50%向上，50%向下！

这不是实验误差。这是量子力学的核心特征。

### 叠加与波函数

z-上的电子可以用数学对象描述：

|ψ⟩ = |↑z⟩

当你测量x轴自旋时，这个状态是x-上和x-下的等量叠加：

|↑z⟩ = (1/√2)|↑x⟩ + (1/√2)|↓x⟩

这不是说"电子是x-上或x-下，我们不知道是哪个"。而是说在测量之前，关于这个问题没有事实存在。电子同时处于两个状态的叠加中。

测量x轴自旋时，波函数"坍缩"到某一个分量——以相应概率。这里是50-50，因为两个振幅的模方相等：|1/√2|² = 1/2。

测量之后，电子处于确定状态。如果得到x-上，新状态是 |↑x⟩。这个状态又是z轴的叠加：

|↑x⟩ = (1/√2)|↑z⟩ + (1/√2)|↓z⟩

所以再测量z轴时，又是50-50。

关键点：**测量x已经使z-自旋随机化了**。不是因为你的装置笨拙地碰了电子——是因为 |↑x⟩ 和 |↑z⟩ 是不同的量子态，互不兼容。

### 数学语言

这套思想可以精确形式化。

**状态**由希尔伯特空间中的向量 |ψ⟩ 描述。对于自旋，这是二维复向量空间，|↑⟩ 和 |↓⟩ 构成一组基。任何状态可以写成：

|ψ⟩ = α|↑⟩ + β|↓⟩

其中 α 和 β 是复数，满足归一化条件 |α|² + |β|² = 1。

**可观测量**（你能测量的物理量）由作用在希尔伯特空间上的数学对象——**算符**——表示。沿z轴的自旋对应算符 Sz，它有两个可能结果（本征值）：+ℏ/2 和 −ℏ/2，对应本征态 |↑⟩ 和 |↓⟩。

**测量**遵循波恩规则：如果系统处于状态 |ψ⟩ = α|↑⟩ + β|↓⟩，测量Sz得到 +ℏ/2 的概率是 |α|²，得到 −ℏ/2 的概率是 |β|²。测量后，状态坍缩到对应本征态。

**演化**在测量之间由薛定谔方程控制：

iℏ ∂|ψ⟩/∂t = Ĥ|ψ⟩

这个方程是线性的、确定性的、时间可逆的。给定某时刻的状态，它唯一确定所有其他时刻的状态。

存在两种不同的演化：大多数时候是平滑确定的（薛定谔方程），测量时是突然概率性的（坍缩）。这种二元性就是**测量问题**的核心。

## 推广：位置、动量、能量

自旋是有限维系统。对于在空间中运动的粒子，希尔伯特空间是无穷维的——波函数 ψ(x) 是位置 x 的复函数。

**波恩规则推广**：在区间 [x, x+dx] 中找到粒子的概率是 |ψ(x)|² dx。

**位置算符** x̂ 通过乘法作用：x̂ψ(x) = x·ψ(x)。它的本征态是 δ 函数，代表完美定域的粒子。

**动量算符** p̂ 涉及微分：p̂ = −iℏ d/dx。它的本征态是平面波 e^(ipx/ℏ)，代表具有完美确定动量的粒子。

**不确定性原理**。位置和动量算符不对易：

[x̂, p̂] = iℏ

这蕴含海森堡不确定性原理：

Δx · Δp ≥ ℏ/2

不存在同时具有精确位置和精确动量的量子态。这不是测量扰动——是根本的不兼容性。

考虑一个完美定域的粒子，波函数是 δ(x−x₀)。它的动量表示（通过傅里叶变换）是所有动量上的常数——完全不确定。反过来，平面波有确定动量，但在整个空间均匀分布。

位置越确定，动量越分散；反之亦然。这是波函数数学结构的直接后果。

**薛定谔方程应用**。对于势能 V(x) 中的粒子：

iℏ ∂ψ/∂t = −(ℏ²/2m) ∂²ψ/∂x² + V(x)ψ

寻找定态（能量本征态）：

Ĥψ = Eψ

对于氢原子（电子处在质子库仑势中），这个方程给出：

En = −13.6 eV / n²,  n = 1, 2, 3, ...

这恰好是 Bohr 临时假设的能级，但现在从第一性原理推导出来。光谱线对应能级间跃迁：

hν = 13.6 eV × (1/n₁² − 1/n₂²)

与实验吻合精确到六位有效数字。原子稳定性之谜解决了：电子处在量子态中，不是经典轨道上。基态（n=1）是能量最低的状态，没有更低能级可以跃迁。

## 奇异后果

### 波粒二象性

回到双缝实验。电子的波函数同时通过两条缝传播：

|ψ⟩ = (1/√2)(|缝1⟩ + |缝2⟩)

干涉图样的出现是因为来自两条路径的振幅在求概率的模方之前先相加。振幅相长的地方，电子聚集；振幅相消的地方，电子稀少。

但当你观察电子走哪条缝时，叠加态坍缩到某一个分量。干涉项消失，你看到两条带。

电子是波还是粒子？答案是：都不是。"波"和"粒子"是从日常经验中派生的经典概念，不适用于量子层面。电子是量子对象，由希尔伯特空间中的波函数描述。它表现出波动行为（干涉）和粒子行为（定域探测），取决于你如何测量。

### 纠缠

单粒子的叠加已经够奇怪。对于多粒子，它导致**纠缠**——粒子之间没有经典类比的关联。

考虑两个电子被制备成总自旋为零（"单态"）：

|ψ⟩ = (1/√2)(|↑↓⟩ − |↓↑⟩)

如果一个沿任意轴自旋向上，另一个必沿同一轴自旋向下。

关键是：这个状态无法分解为每个电子的独立状态。它们是纠缠的——联合状态无法写成单粒子态的乘积。

现在把电子分开任意距离。测量电子1沿某轴的自旋，假设得到"上"。瞬间，你知道电子2沿同一轴将给出"下"——确定无疑。

乍看像普通关联：手套成对，发现右手手套就知道另一个是左手。Einstein、Podolsky 和 Rosen（EPR）在1935年论证说，电子必定携带决定结果的隐藏信息。

但1964年，John Bell 证明这行不通。他推导出任何"定域隐变量"理论都必须满足的不等式。关键假设：
1. 物理性质在测量前有确定值
2. 这里的测量不影响那里的结果

量子力学预测的关联违反贝尔不等式。实验已经确凿证实量子力学（2022年诺贝尔奖）。自然不允许定域隐变量。

然而纠缠不能传递信息。在比较结果前，两个观察者看到的都只是随机结果——50%上，50%下。只有核对记录时，关联才显现。

纠缠到底意味着什么仍有争议。但它的真实性毋庸置疑，已被应用于量子计算、量子密码和量子隐形传态。

## 诠释之争

标准（哥本哈根）诠释说：测量时波函数坍缩，引入真正的随机性；观测在物理上扮演特殊角色。

但存在严肃的替代方案。

**多世界诠释**（Hugh Everett，1957）：不存在坍缩。波函数始终按薛定谔方程演化。当测量发生时，观察者与系统纠缠，分裂成多个版本——每个版本感知到不同结果。所有结果都在现实的不同"分支"中发生。量子力学本质上是确定性的；表面随机性是我们身处某一分支内的视角效应。

优雅，但意味着无数个分裂的你自己和宇宙。

**玻姆力学**：粒子始终有确定位置。波函数引导粒子运动。随机性源于对初始位置的无知，像经典统计力学。精确重现所有量子预测。

代价是显式非定域性：一个粒子的轨迹瞬时依赖于所有其他粒子的位置。

没有哪种诠释获得共识。在经验上，它们目前无法区分（除了某些可能在未来检验的修改理论）。在它们之间的选择由关于实在、定域性和确定性的哲学偏好驱动。

## 应用

量子力学不仅是哲学。它是现代技术的基础。

**半导体与电子学**。晶体管利用晶体中电子能级展宽成能带的量子效应。每块计算机芯片、每部智能手机的核心都依赖量子力学。

**激光**。Einstein 在1917年推导出原子可以被激励发射与入射光子相同的光子——"受激发射"。这使相干光放大成为可能，应用于通信、手术、制造。

**量子计算**。经典比特是0或1。量子比特是叠加态 α|0⟩ + β|1⟩。n个量子比特可存在于2^n个基态的叠加中，实现指数级并行。Shor算法威胁当前加密。分子的量子模拟可能彻底改变化学。

实用的量子计算机仍然有限——维持相干性极其困难——但它们代表根本上新的计算范式。

## 未解之谜

量子力学取得了惊人成功，但基础问题仍未解决。

**测量问题**。坍缩何时以及如何发生？什么构成"测量"？如果一切（包括测量装置）都由按薛定谔方程演化的量子物质构成，坍缩从何而来？这不是实践问题——量子力学在计算上完美工作。它是概念上的裂缝：两种演化规则，没有明确指导何时用哪种。*《测量问题》（第47章）深入探讨这个难题。*

**量子引力**。广义相对论把引力描述为时空弯曲。量子力学把物质和力描述为量子场。将它们结合极其困难。量子化引力可能需要量子化时空本身。弦理论和圈量子引力是主要方向，但都未完成，也未经实验证实。

**为什么是量子力学？**为什么是复希尔伯特空间？为什么是线性算符？为什么是波恩规则？量子力学是为匹配实验而构建的，不是从第一性原理推导的。许多物理学家感到我们的理解中缺少了什么。

## 总结

量子力学用根本不同的东西取代了经典现实图景：

1. **状态**由波函数表示，编码概率振幅而非确定性质
2. **叠加**允许状态成为经典上截然不同可能性的组合
3. **测量**产生本征值，概率由 |⟨a|ψ⟩|² 给出，引入坍缩
4. **不确定性原理**限制对不对易可观测量的同时了解
5. **纠缠**创造出违反贝尔不等式的关联
6. **演化**在测量间是确定的（薛定谔方程），测量时是概率的（坍缩）

这个框架在数学上精确，在经验上无比成功。但它意味着什么——告诉我们关于现实本质的什么——仍深深不确定。

这种不确定性不是失败，而是反映了深层难题：可能需要新物理学、新哲学，或两者兼备。

## 延伸阅读

**Feynman, Leighton 和 Sands. 《费曼物理学讲义》第三卷.** 传奇讲座，强调物理直觉。免费：feynmanlectures.caltech.edu

**Maudlin, Tim. 《物理学哲学：量子理论》（2019）.** 对基础问题和诠释的清晰哲学分析。
